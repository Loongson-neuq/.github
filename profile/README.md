## 关于作业

### 如何完成作业？

1. 准备网络环境，创建 Github 用户： https://github.com 。

2. 点击作业链接，会返回一个仓库链接，使用 Git 克隆仓库至本地。

3. 按照仓库中的 `README.md` 完成作业，使用 Git 提交并推送。

> 遇到问题请参考 [Git 教程](https://manual.caiyi1.me/use-git)
